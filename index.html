HTML positioning

Placing HTML elements where you want is not always easy. The first thing you should know, is the CSS Box model.

Margin

Border

Padding

Content

The following 3 values are essential to use to position HTML elements:

Property Usage Description
Margin margin: 10px; The margin defines how much space there must be between the outside of the element, to the elemtent beside.
Border border: 1px solid red; The border defines the lines on the outside of the padding and the content. A border can be colered and set to different styles.
Padding padding: 20px; The padding defines how much space there must be between the border and the content.

There are more ways to speficy the arguments of these than specified in the table. For example, if two inputs are given to the margin property, the top and margin would be the first input and the left and right would be the second input.
