Standard Styles

Size does matter! The following shows the standard styles and sizes for some important HTML elements:

The H1 Tag:

  • Font size: 50px
  • Font family: Roboto, sans-serif
  • Font weight: 700 (bold)

The H2 Tag:

  • Font size: 30px
  • Font family: Roboto, sans-serif
  • Font weight: 700 (bold)

The H3 Tag:

  • Font-size: 22px
  • Font family: Roboto, sans-serif
  • Font weigth: 700 (bold)
  • Color: #6e6c6c

The P Tag:

  • Font size: 18px
  • Font family: Roboto, sans-serif
  • Font weight: 400 (regular)
